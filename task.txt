1. Создать фреймворк для автоматизированного тестирования GUI сайта 
Мебель март  (https://mebelmart-saratov.ru/myagkaya_mebel_v_saratove/divanyi_v_saratove), который должен удовлетворять следующим требованиям:

1.1 реализует паттерн Layered Architecture Pattern;
1.2 использует систему отчетности Allure;
1.3 при фейле теста делает скриншот экрана.
1.4 реализация фреймворка должна вестись с использованием GIT системы, результаты работы должны быть опубликованы в GIT репозитории github / gitlab.
1.5 использовать явные ожидания, если необходимо.
1.6 поддерживает работу с двумя браузерами Google Chrome, Firefox.
1.7 реализует Pattern Page Object для страниц сайта.
1.8 реализует доступ к обьектам страницы через CSS или XPath локаторы.

2. Реализовать следующие тестовые сценарии:

2.1 Фильтрация по характеристикам и проверка наличия товара
Шаги:
• Открыть раздел «Диваны» (https://mebelmart-saratov.ru/myagkaya_mebel_v_saratove/divanyi_v_saratove).
• В фильтре "Цена" установить диапазон, например, от 10 000 ₽ до 15 000 ₽ (или применить другой доступный фильр, например, по ширине, если он есть).
• Применить фильтр.
• Найти в результатах выдачи диван с конкретным названием или артикулом, который должен попадать под этот фильтр (например, ориентируясь на цену из примера на странице). 
Ожидаемый результат: В выдаче присутствует товар с указанной ценой, соответствующей условиям фильтрации.

2.2 Проверка деталей товара в карточке
Шаги:
• Открыть раздел «Диваны».
• В списке товаров найти и кликнуть на диван, например, «Диван Бостон» (нужно выбрать конкретную модель из каталога).
• На странице товара проверить наличие и корректность характеристик, например, ширины (Ширина: 2100 мм).
• Ожидаемый результат: Открылась карточка правильного товара, и указанная характеристика (например, ширина) совпадает со значением, которое было в общем списке товаров.

2.3 Добавление товара в избранное
Шаги:
• Открыть раздел «Диваны».
• У первого товара в списке нажать иконку «Избранное» (♥).
• Перейти в раздел «Избранное» (иконка или пункт меню).
• Ожидаемый результат: Выбранный диван отображается в списке избранного. Иконка «Избранное» у этого товара в каталоге изменила свое состояние (например, закрасилась).

2.4 Поиск товара по названию
Шаги:
• На любой странице сайта ввести в поисковую строку запрос, например, «Бостон» (часть названия существующего дивана).
• Нажать кнопку «Найти» или клавишу Enter.
• Ожидаемый результат: В результатах поиска первым отображается товар, в названии которого есть слово «Бостон».

2.5 Добавление товара в корзину и проверка стоимости
Шаги:
• Открыть раздел «Диваны».
• У первого товара в списке нажать кнопку «В корзину» или иконку корзины.
• Перейти в «Корзину» (обычно иконка вверху страницы).
• Ожидаемый результат: В корзине находится добавленный товар. Его цена соответствует цене, указанной в каталоге. 

Если на сайте есть другие значимые разделы (например, «Кресла»), логику кейсов можно легко перенести и на них. Нужно будет только уточнить актуальные названия фильтров или конкретных товаров на странице.
